Our Solution

In our application, we implemented various features that could be easily understood and could be utilized by both new and existing users. In the application a sign-up form is generated for new users, login form for existing users, a product page that displays the available products in stock and users can decide on what products they desire to place in either their shopping cart or shopping list. A shopping list is where users can store item for future or current use and it is designed to assist the users with their spending plan. Additionally, a shopping list can be converted into a shopping cart.

We also implemented a recommendation system which works by using ratings given by users to items, and grouping users based on these ratings. Recommendations are then provided to users for items they have not rated, but their fellow users have.

Technologies

The VueJs framework was used to build and serve our User Interface. It allows for incremental adjustments and improvements to be made so our project is open to more features and can keep getting better as time goes by.


It was also coded in Python so our project is easy to facilitate maintainable development and allowed for different approaches to be employed in generating our solution. Along side this the flask framework was used to create the functionality for our software solution, it was easy to understand and implement.

The Future

For future implementation we would like to implement features to allow for offline use. In addition to this we would like to make users able to access our application via mobile on the platform of their choice (Google Play or Apple IOS Store). We would also like the application to facilitate a courier system that would be composed of registered users (whether companies themselves or lone users), who would be able to establish their profile as such (being an existing user, they would then sign up to self employ as a courier). Eventual inclusion of restaraunts and other places that would like to market thier wares via our app(wholesale or otherwise)..
